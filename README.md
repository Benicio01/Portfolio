# Portfolio - Benicio Braunstein

Portfolio personal desarrollado con **Astro** y **Tailwind CSS**.

## 🚀 Tech Stack

- **Framework:** Astro
- **Estilos:** Tailwind CSS v4
- **Hosting:** Vercel / Netlify

## 🛠️ Tecnologías

### Lenguajes
- Python
- Rust
- Java
- SQL
- React
- JavaScript

### Herramientas
- Git
- Linux
- Docker
- VS Code
- PostgreSQL
- Node.js

## 📦 Setup

```bash
# Instalar dependencias
npm install

# Desarrollo
npm run dev

# Build para producción
npm run build
```

## 🌐 Deploy

Deploy automático en **Vercel**:
1. Importar repo en [vercel.com](https://vercel.com)
2. Deploy automático al hacer push a main
